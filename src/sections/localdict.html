create local dict:

Select directory. Pecha.js will scan (recursively) the directory for files containing the Tibetan text. Any plain-text or html file (utf-8) goes ok. The files can contain text of any languages, only Tibetan text will be processed. But binary file formats (.doc, .zip, .pdf, etc) will not be processed.

Files "localDict.csv" and (possible) "unprocessed.csv" will be created in this directory

note: scanning and analyzing process can take some time!

then:

1. edit new CSV dictionary

2. import ready CSV to Pecha.js, look around

3. repeat points 1. and 2. as many times as needed

4. publish your new awesome dictionary to web and github (disabled untill authentication on server will work)
